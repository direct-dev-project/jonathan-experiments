# Jonathan's Experiments

Experimental tools, dashboards, and utilities built by Jonathan for Direct.dev.

## Projects

### sync-dashboard
Real-time monitoring dashboard for E2E sync tests. Visualizes drift, latency, and speedup metrics.

- **Run**: `cd sync-dashboard && npm install && npm start`
- **Access**: http://localhost:3000
- **Data source**: `/tmp/e2e-sync-data.ndjson`
